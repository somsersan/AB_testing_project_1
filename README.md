# Кейс: 
На Wildberries начали пристальнее следить за своим ценообразованием, из-за чего решили обучить ML модель для построения кривых эластичности СПП*, чтобы более грамотно управлять скидками.

Команда ценообразования разработала ML модель и начала её обучать: они выбрали пользователей в 7 групп (1 контроль, и 6 тестов) и запустили модель. Предварительно они не делали никаких статистических проверок, т.к. цель - поднабрать онлайн данные для обучения моделей.

К вам пришли как к аналитику с просьбой посмотреть и оценить результаты текущих моделей. 

# Задача: оценить результаты моделей

* mde.parquet - исторические данные по заказам пользователей теста для расчёта mde
* homogenuity.parquet - данные с пользовательскими метриками для оценки однородности групп
* result.parquet - данные, собранные по окончании теста для оценки эффекта
