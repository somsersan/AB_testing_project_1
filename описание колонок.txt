`mde.parquet`

	- `grp` - название группы пользователей:
    		* control - контрольная группа
   		* Xprc_increase - снижении СПП на X%
    		* Xprc_decrease - увеличение СПП на X%
	- `user_id` - ID пользователя
	- `bu` - Бизнес Юнит  
	- `parent_name` - Родительская категория
	- `subject_name` - Предмет (подкатегория)
	- `orders_sum`- сумма заказов
	- `gmv_sum` - сумма выкупа
	- `pre_week` - неделя предпериода

`homogenuity.parquet`

	- `user_id` - ID пользователя
	- `grp` - название группы пользователей:
    		* control - контрольная группа
    		* Xprc_increase - снижении СПП на X%
    		* Xprc_decrease - увеличение СПП на X%
	- `bu` - Бизнес Юнит  
	- `parent_name` - Родительская категория
	- `subject_name` - Предмет (подкатегория)
	- `created_orders_cnt` - кол-во созданных заказов
	- `created_items_cnt` - кол-во заказанных товаров
	- `created_items_rub` - стоимость созданных заказов
	- `bought_items_cnt` - кол-во выкупленных товаров
	- `bought_items_rub` - стоимость выкупленных товаров
	- `spp_rub` - стоимость выделенного СПП
	- `spp_pct` - средний % спп
	- `order_days_cnt`- кол-во дней с заказами
	- `aiv` - средняя стоимость заказанного товара
	- `aov` - средняя стоимость созданного заказа 
	- `upt` - среднее кол-во товаров в заказе
	- `age` - возраст
	- `gender` - пол
	- `profit` - прибыльность пользователя для площадки
	- `purchase_percent` - процент выкупа
	- `purchase_sum` - сумма выкупа
	- `locale` - локаль
	- `region` - регион
